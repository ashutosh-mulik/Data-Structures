# Visual Data Structures

### Aim 
- To give a visual introduction to the world of Array, Stack, Queue, Linked List.

### Technology
- Python (Tkinter)

### Features 
1. Open-Source 
2. Cross platform
3. Includes Array, Stack, Queue, Linked List
